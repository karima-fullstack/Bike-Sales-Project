# 🚴 Bike Sales Excel Dashboard

## 📌 Project Overview

This project is an end-to-end Excel data analysis and dashboarding project based on a bike buyers dataset.

The objective was to analyze customer demographics and purchasing behavior to identify the factors that influence bike purchases.

The project demonstrates skills in:

* Data Cleaning
* Data Transformation
* Pivot Tables
* Pivot Charts
* Dashboard Design
* Data Visualization
* Excel Reporting

---

## 📊 Dataset

The dataset contains customer information including:

* Marital Status
* Gender
* Income
* Number of Children
* Education
* Occupation
* Home Ownership
* Number of Cars
* Commute Distance
* Region
* Age
* Bike Purchase Status

---

## 🧹 Data Cleaning Process

The raw dataset was cleaned and transformed before analysis.

### Transformations Performed

#### 1. Marital Status Standardization

Converted abbreviations:

* M → Married
* S → Single

#### 2. Gender Standardization

Converted abbreviations:

* M → Male
* F → Female

#### 3. Age Group Creation

Created a new field called **Age Brackets** to categorize customers into age groups.

Example:

* Adolescent
* Middle Age
* Old

#### 4. Data Validation

Verified data consistency and prepared the dataset for analysis.

---

## 📈 Data Analysis

Several Pivot Tables were created to analyze customer behavior.

### Analysis Performed

#### Income vs Bike Purchase

Analyzed average income of customers who purchased bikes compared to those who did not.

#### Customer Age Analysis

Studied purchasing behavior across age groups.

#### Commute Distance Analysis

Examined how commuting distance affects bike purchasing decisions.

#### Regional Analysis

Compared bike purchases across different regions.

---

## 📊 Dashboard

An interactive dashboard was built using:

* Pivot Tables
* Pivot Charts
* Slicers

The dashboard allows users to filter and explore data dynamically.

Dashboard filters include:

* Marital Status
* Region
* Education
